# Website Resmi Polda Papua Tengah

> Official Website of Kepolisian Daerah Papua Tengah

---

## Project Overview

Website Resmi Polda Papua Tengah merupakan portal informasi publik yang dibangun menggunakan React, TypeScript, dan Vite.

Website ini hanya berfungsi sebagai Frontend Public Website.

Seluruh konten website berasal dari CMS melalui REST API.

---

## Project Architecture

Project terdiri dari dua aplikasi yang terpisah:

1. Website Frontend
2. CMS (Content Management System)

Frontend hanya membaca data dari CMS melalui API.

---
